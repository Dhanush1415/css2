CSS-Cascading Style Sheet

Quick intro -

The css which stands for Cascading Style Sheet, which is widely used by the programers in wep page designing and app development ,to make the Webpage or the Application look better for the client.

Types:
  • Inline css
  • Internal css
  • External css

Inline css

An Inline css which is used in the HTML opening tag as like an attribute.->

Internal Css

An Internl Css which is used inside the head tag with a "style" tag inside the style tag the css styles were written in the details of the element name i.e h1,p,div,etc or the selector name as the class name.

External css

An External css is which the styles are writen in the .css program file in a external field and linked to the html program in the head tag which should be ececuted where the program link is linked .
